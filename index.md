### **How to set up overlay**

**Required:**
- node.js
- bakkesmod sos plugin
- sos-ws-relay
- Enviroment Variable that points to bakkesmodSDK

**Links:**
- [Bakkesmod sos plugin (requires manual install)](https://gitlab.com/bakkesplugins/sos/sos-plugin/-/releases)
